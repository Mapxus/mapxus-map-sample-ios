# CHANGELOG

## v8.21.0 (2026-06-22)

### 📝 Changes

* The SDK version has been upgraded to 8.7.0
* Expanded Multilingual Support
  * Full SDK support added for French and Italian:
    * Map Display: Users can set the interface language to view map content in French or Italian
    * Search Functionality: Venue, Building, and POI search results now include dedicated French/Italian text fields
    * Navigation Instructions: Turn-by-turn navigation instructions are now available in French and Italian
* Enhanced Navigation Capabilities
  * Added a new `elevator` navigation mode: Route planning will prioritize elevators as the primary method for floor/space transitions, catering to accessibility needs or similar use cases.
